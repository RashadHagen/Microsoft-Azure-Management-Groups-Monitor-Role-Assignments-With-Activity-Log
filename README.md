<h1>Microsoft Azure Management Groups – How To Monitor Role Assignments With Activity Log</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
In this project, Microsoft Azure Management Groups will be used to observe and manage subscription-level events related to access control by reviewing role assignment activity within the Azure environment. You will monitor Role-Based Access Control (RBAC) role assignments at the management group level using the Azure Activity Log to gain visibility into changes affecting subscriptions. You access the Azure portal, locate Management Groups through the global search bar, select a specific management group, and open the Activity Log blade positioned between Resources and Access control (IAM) to review recorded subscription-level events related to role assignments. This process enables you to view and track RBAC role assignment activity through the Activity Log, providing clear insight into access-related changes occurring across subscriptions within the selected management group.
